## 🚀 Overview

A modern **Executive Analytics Dashboard** built with **Streamlit** that provides real-time insights into business performance using sales data.

This dashboard simulates **enterprise-level BI tools** with:

* Dynamic filtering
* KPI tracking
* Animated metrics
* Interactive visualizations

---

## ✨ Features

* 🎛️ **Interactive Filters**

  * Year selection
  * Region filtering
  * Category filtering

* 📊 **KPI Metrics**

  * Revenue
  * Profit
  * Profit Margin
  * Revenue Growth

* ⚡ **Animated KPI Cards**

  * Smooth number transitions
  * Glassmorphism UI design

* 📈 **Advanced Visualizations**

  * Monthly revenue flow (area chart)
  * Profit distribution (donut chart)
  * Regional performance (grouped bar chart)
  * Top products (horizontal bar chart)

* 📉 **Business Insight Alerts**

  * Revenue growth/decline indicator
